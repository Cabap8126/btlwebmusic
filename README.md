# 🎵 BTL Web Music Project

## 📌 Giới thiệu
Dự án web quản lý nhạc (Music Management System) được xây dựng bằng Node.js, Express và MongoDB.

Hệ thống cho phép quản lý các thực thể chính như:
- 🎶 Song (Bài hát)
- 🎤 Singer (Ca sĩ)
- 🏷️ Topic (Chủ đề)

---

## 👨‍👩‍👧‍👦 Thành viên nhóm & phân công

### 1. Trần Đình Mạnh
- 📦 Module: Song
- 📁 File chính: `song.js`
- 🔥 Chức năng:
  - Create Song
  - Read Song
  - Update Song
  - Delete Song (soft delete)

---

### 2. Phạm Việt Khoa
- 📦 Module: Singer
- 📁 File chính: `singer.js`
- 🔥 Chức năng:
  - Create Singer
  - Read Singer
  - Update Singer
  - Delete Singer

---

### 3. Ngô Trọng Mạnh
- 📦 Module: Topic
- 📁 File chính: `topic.js`
- 🔥 Chức năng:
  - Create Topic
  - Read Topic
  - Update Topic
  - Delete Topic

---

## 🧠 Công nghệ sử dụng

- Node.js
- Express.js
- MongoDB
- Mongoose
- Nodemon

---

## ⚙️ Chức năng hệ thống

### 🎵 Song Management
- Thêm bài hát mới
- Xem danh sách bài hát
- Cập nhật thông tin bài hát
- Xóa bài hát (soft delete)

### 🎤 Singer Management
- Quản lý ca sĩ
- CRUD đầy đủ

### 🏷️ Topic Management
- Quản lý chủ đề nhạc
- CRUD đầy đủ
